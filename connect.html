Connecting to cluster

In order to connect to cluster, you must use an SSH client. See below for instructions based on your operating system.

On Campus or CAM VPN Host URL

While connected to most networks at the University of Connecticut or Univeristy of Connecticut Health center or if you are connected to the Cam VPN, the cluster can be accessed through this host url:

mantis-submit.cam.uchc.edu

This host is planned to be phased out in the near future so that only the external host explained below will be used.

External Host URL

The cluster can also be accessed through login.hpc.cam.uchc.edu from any network (on/off campus, with/without VPN) but requires that you setup an ssh key.

Using SSH Client

  • Mac and Linux Users
  • Windows Users

An ssh client is installed by default on Mac and Linux operating sytems and can be run from the Terminal App.

Connecting

To create a connection, enter the ssh command followed by your username and host url like so. Login with

ssh <user>@<host>
